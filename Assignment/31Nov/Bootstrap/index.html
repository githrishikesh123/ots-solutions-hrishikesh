2. jQuery is removed completely: ==========> Earlier, bootstrap components like modals, tooltips, popovers etc, were dependent on jQuery and popper.js. After the release of the alpha 5 version, Bootstrap no longer depends on jQuery. While jQuery has been used in Bootstrap for more than 8 years, many developers had this issue that jQuery brought complex JavaScript behaviors to them. 3. Switch to Vanilla JavaScript:==========> One of the major reasons of dropping jQuery was to redefine the framework completely on modern JavaScript standards. 4. Dropped support of Internet Explorer 10 and 11: ============> When bootstrap decided to refine their framework purely on Vanilla JavaScript, they dropped the support of IE which is not compatible with modern JavaScript standards.
